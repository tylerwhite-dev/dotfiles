## basic setup

### name
```
git config --global user.name "..."
```

### email
```
git config --global user.email "..."
```

## save passwords in git
```
git config --global credential.helper store
```

## git-lfs
install git-lfs & go to repository
```
git lfs install
git lfs fetch
git lfs checkout
```

## git submodule
```
git submodule init
git submodule update
```
