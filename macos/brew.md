## Configurations

### Starship

add to file ~/.zshrc
```
eval "$(starship init bash)"
```

### zsh-autosuggestions

To file ~/.zshrc
```
source /opt/homebrew/share/zsh-autosuggestions/zsh-autosuggestions.zsh
```
